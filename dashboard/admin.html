Configuracion de Prueba de Vida

Define el modo de liveness para cada suscriptor. El campo combina tipo de disparo y prueba de vida en un solo concepto.

Modo Disparo Prueba de Vida Uso Recomendado
Sonrisa Automatico (happy ≥ 0.95) La sonrisa ES la prueba de vida Acceso rapido, kioscos
Sorpresa Automatico (surprised ≥ 0.90) La sorpresa ES la prueba de vida Pruebas alternativas
Boca Abierta Automatico (mouth ≥ 0.85) Abrir boca ES la prueba de vida Verificacion controlada
Mov. Cabeza Automatico despues de mover cabeza Mover cabeza izq/derecha Alta seguridad
Boton + Gesto Manual (usuario presiona) El usuario realiza un gesto y presiona boton Control total, accesibilidad
Boton + Cabeza Manual (usuario presiona) El usuario mueve cabeza y presiona boton Control total + seguridad
